    class="text-center": centers text

    Button
  • class="btn btn-default": Default button
  • class="btn btn-block": Bootstrap button that takes 100% available width
  • class="btn-primary": Adds color to button that indicates desired action
  • class="btn-info": Adds color to button that indicates optional action
  • class="btn-danger": Adds color to button that indicates destructive action

    jQuery

    $(document).ready(function(){}); Guarentees that script does not run until HTML has finished loading.
    $("selector").addClass("animated bounce"); shake fadeout
    .removeClass: Takes away class from element.
    .css("attribute", "value");: Change value of CSS attribute.
    .prop(): Modifies properties of elements ("disabled", true);
    .html(): Replaces element referred to by the selector.
    .text(): Same as above, but interprets tags as text.
    .remove(): Removes element.
    .appendTo("target_element): Moves an element to another container element.
    .clone(): Copies an element.
    .parent(): Gets the parent HTML element.
    .children(): Gets the children of an HTML element.
    :even/:odd/:nth-child(n): Selects all the even/odd/nth element.
    Function chaining: Using the output of a function as the argument of another function e.g. clone().appendTo()

    Javascript

    Comments: // for single line; /* */ for multi-line.
    =: Assignment operator
    Variable names should be in camelCase
    Increment operator: myVar++ is the same as myVar = myVar + 1;
    Remainder operator: 5%2=1
    Operate and Assign: +=, -=, *=, /=
    Escape character: \"
  • \: \\
  • newline: \n
    • Single quotes are used the same way as double quotes; use one to avoid having to escape the other.
    Concatenation of strings also use +
    Bracket notation: zero-based - [0] = the first character in the string
    String variable is immutable: individual chars cannot be changed, can only change whole string.
