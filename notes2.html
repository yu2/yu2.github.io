    Comments: // for single line; /* */ for multi-line.
    =: Assignment operator
    Variable names should be in camelCase
    Increment operator: myVar++ is the same as myVar = myVar + 1;
    Remainder operator: 5%2=1
    Operate and Assign: +=, -=, *=, /=
    Escape character: \"
  • \: \\
  • newline: \n
    • Single quotes are used the same way as double quotes; use one to avoid having to escape the other.
    Concatenation of strings also use +
    Bracket notation: zero-based - [0] = the first character in the string
    String variable is immutable: individual chars cannot be changed, can only change whole string.
    Array: var myArray = ["item1", "item2"];
    Multi-dimensional Array: var myArray = [["Monday", 1],["Tuesday", 2]];
    Array entries are mutable: Can be changed through index

  • .unshift(): Append to the beginning of an array.

  • Equality: ==
