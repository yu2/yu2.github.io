Solitude

              By Ella Wheeler Wilcox
              
              Laugh, and the world laughs with you;
              Weep, and you weep alone;
              For the sad old earth must borrow its mirth,
              But has trouble enough of its own.
        
              Sing, and the hills will answer;
              Sigh, it is lost on the air;
              The echoes bound to a joyful sound,
              But shrink from voicing care.
        
              Rejoice, and men will seek you;
              Grieve, and they turn and go;
              They want full measure of all your pleasure,
              But they do not need your woe.
              
              Be glad, and your friends are many;
              Be sad, and you lose them all,-
              There are none to decline your nectared wine,
              But alone you must drink life's gall.
          
              Feast, and your halls are crowded;
              Fast, and the world goes by.
              Succeed and give, and it helps you live,
              But no man can help you die.
        
              There is room in the halls of pleasure
              For a large and lordly train,
              But one by one we must all file on
              Through the narrow aisles of pain.

HTML Notes

<ul><li></li></ul>: unordered list
<ol>: ordered list
<a href=""></a>: link
target="_blank": open link in a new tab
<img src="" />: image
  • alt: description of image to be displayed if image fails to load and for visually-impaired users
<!-- -->: comment

CSS

<style> element must be placed within the head
link to css file: <link href="style.css" type="text/css" rel="stylesheet">
property + value = declaration
selector(s) + declaration(s) = rule
comments: /* */
named colours, RGB colours
  • rgb(a), hex, hsl(a)
serifs better for long paragraphs?
link to fonts to avoid users not having them installed
increase leading with line-height to make more readable
pixels for precise control, ems for fluid layout
line-height, word-spacing, letter-spacing, font-weight(100-900, normal, bold, italic), text-transform, text-align
selecting elements
  • #(octothorpe) is used to select id
  • . is used to select class
  • multiple classes can be assigned to one element to further categorize
  • element.class selector

Box Model

everything is a box
  • ems of a box: relative to the size of the text within
  • %: relative to the size of the box it is enclosed in
Responsive web design min-width and max-width; min-height and max-height: minimum and maximum size of elements to ensure they don't appear too small or large on screens of different sizes
border-style(solid, dashed, grooved, etc), border-width(px, clockwise from top to set each border), set individually with border-top-width, border-right-width, border-left-width, border-bottom-width
shorthand to reduce code bloat: border: width style color; border-radius
