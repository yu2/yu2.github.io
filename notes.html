HTML Notes

<ul><li></li></ul>: unordered list
<ol>: ordered list
<a href=""></a>: link
target="_blank": open link in a new tab
<img src="" />: image
  • alt: description of image to be displayed if image fails to load and for visually-impaired users
    • <!-- -->: comment

    CSS

    <style> element must be placed within the head
    Link to css file: <link href="style.css" type="text/css" rel="stylesheet">
    Property + value = declaration
    Selector(s) + declaration(s) = rule
    Comments: /* */
    named colours, RGB colours
  • rgb(a), hex, hsl(a)
    • Serifs better for long paragraphs?
    Link to fonts to avoid users not having them installed
    Increase leading with line-height to make more readable
    Pixels for precise control, ems for fluid layout
    line-height, word-spacing, letter-spacing, font-weight(100-900, normal, bold, italic), text-transform, text-align
    Selecting elements
  • #(octothorpe) is used to select id
  • . is used to select class
  • multiple classes can be assigned to one element to further categorize
  • element.class selector

    • Box Model

    Everything is a box
  • ems of a box: relative to the size of the text within
  • %: relative to the size of the box it is enclosed in
    • Responsive web design
    min-width and max-width; min-height and max-height: minimum and maximum size of elements to ensure they don't appear too small or large on screens of different sizes
    overflow (hidden, scroll): Controls what happens to content when it spills outside of its box
    border-style(solid, dashed, grooved, etc), border-width(px, clockwise from top to set each border), set individually with border-top-width, border-right-width, border-left-width, border-bottom-width
    Shorthand to reduce code bloat: border: width style color;
    border-radius: set corners to curvature equal to a circle's radius, in px or percentage
    padding: space between content of box and its borders (top, right, bottom, left) or (top+bottom, right+left)
    margin: space outside of the box (distance away from the next box)
  • Can also set margin of individual sides with margin-top, margin-right, etc
    • Set magin and padding to zero to override browser's default user agent stylesheet.
    display changes how elements are displayed - inline, block, inline-block, or none.
  • Can be used to make li items into navigation bar.
    • visibility: hidden hides the content of the element, but the space reserved for it (its box) will remain.
    Actual rendered size: width/height + padding + border
  • box-sizing: border-box (not affected by border thickness or padding

    • Positioning

    Positioning:
  • static:default
  • relative: relative to default position (set with top, right, bottom, left)
  • absolute: other elements ignore it
