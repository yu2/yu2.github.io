Constructor function: Used to create objects
  • Define a Constructor function: var Car = function() {this.wheels = 4;}
  • Use the Constructor to create a new instance: var myCar = new Car();
  • Use parameters to assign values: var Car = function(wheels) {this.wheels = wheels;}
  • Define private properties with var privateProp = 1;
  • Public methods to change private properties: this.doSomething = function(something) {};
    • Array functions
  • Map: Iterates over an array to produce a new array.
  • var arrMap = oldArray.map(function(val){return val + 3;});
  • Reduce: Iterates through an array two items at a time, to condense the array into a single value
  • Filter: Iterates through an array, taking out members that don't satisfy a condition
  • Sort: Sorts an array depending on smallest to largest (a - b), or largest to smallest (b - a)
  • Reverse: myArray.reverse();
  • Concat: newArray = oldArray.concat(arrayToTheEnd);
  • .slice(start [,end]): Index to begin slice; index before end of slice
  • .splice(beginIndex [, deleteCount] [,item1, ...]): Change array by removing elements/adding new ones.
  • .filter(word ==> word.length > 6): Filter out items in an array that don't satisfy the criteria.
    • String functions
  • Split a string into an array: string.split("delimiter");
  • Join elements of an array into a string: myArray.join("delimiter");
  • str.charCodeAt(index): Returns utf-16 code of unit at index.
  • String.fromCharCode(charCode): Returns str corresponding to utf-16 values.
    • .replace: Replace regular expression matches with second argument.
    Boolean constructor creates an object with an initial value of true for "truthy" values and false for falsy values (false, null, 0, "", undefined, and NaN). Arguments Object: Array-like Object that can be used to access variable arguments.
  • Convert arguments object to array with Array.prototype.slice.call(arguments);

    • JSON APIs and Ajax

    HTTP: HyperText (text with links in it) Transfer Protocol (rules for gettings things from one place to another e.g. webpages to browser)
  • A client makes a HTTP request, the internet finds a server that fulfills the request and sends a response
  • Clicking links makes a state transition and brings about next state (Representational State Transfer).
    • API (Application Programming Interface): Contract for how a program can interact with an application.
    XML Http Request
  • new XMLHttpRequest()
  • xhr.open("GET", "https://www.codecademy.com/", false);
  • GET (retrieve), POST (send), PUT (update information), DELETE
    • Endpoints: API-defined locations where particular data are stored.
