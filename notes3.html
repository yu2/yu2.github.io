Object-Oriented and Functional Programming

Constructor function: Used to create objects
  • Define a Constructor function: var Car = function() {this.wheels = 4;}
  • Use the Constructor to create a new instance: var myCar = new Car();
  • Use parameters to assign values: var Car = function(wheels) {this.wheels = wheels;}
  • Define private properties with var privateProp = 1;
  • Public methods to change private properties: this.doSomething = function(something) {};
    • Array functions
  • Map: Iterates over an array to produce a new array.
  • var arrMap = oldArray.map(function(val){return val + 3;});
  • Reduce: Iterates through an array two items at a time, to condense the array into a single value
  • Filter: Iterates through an array, taking out members that don't satisfy a condition
  • Sort: Sorts an array depending on smallest to largest (a - b), or largest to smallest (b - a)
  • Reverse: myArray.reverse();
  • Concat: newArray = oldArray.concat(arrayToTheEnd);
  • Split a string into an array: string.split("delimiter");
  • Join elements of an array into a string: myArray.join("delimiter");
    • Replace: Replace regular expression matches with second argument.
